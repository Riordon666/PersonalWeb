# Riordon's Blog v1.0 更新说明与网站介绍

今天想认真写一篇"里程碑"——本网站（博客）正式进入 **v1.0**。

从最初的想法到现在能稳定地写作、展示、切换主题、在移动端也保持良好体验，这个过程里做了很多小优化。v1.0 不代表终点，而是一个"可以放心持续迭代"的起点。

## 这是什么网站？

这是我的个人网站 + 博客。

整个站点由两部分组成：

### 起始页（首页）

访问根路径 `/` 时，你会看到一个**水墨画风格**的起始页：

- **动态水墨背景**：Canvas 绘制的流动水墨粒子效果，营造沉浸式氛围。
- **贪吃蛇动画**：背景中有一条贪吃蛇在游走，增添趣味。
- **打字机效果**：副标题"心之所向，素履以往"以打字机方式逐字显示。
- **平滑过渡**：点击 `enter` 按钮后，页面以波浪形 SVG 路径动画过渡到主卡片区域。

### 博客系统

访问 `/blog/` 进入博客，你可以在这里看到：

- **文章**：学习笔记、项目复盘、随手记录。
- **归档**：按时间线整理的所有文章。
- **标签**：按主题聚合文章。
- **主题系统**：支持主题色与背景图（含模糊/透明度），并可在不同页面间保持一致。

## v1.0 版本我做了哪些更新？

这次更新主要聚焦在"读起来更舒服、用起来更顺手"。

### 1. 移动端体验优化

- **侧边栏交互更自然**：切换页面后不会莫名其妙自动展开；点击导航项会自动收起。
- **文章目录（TOC）移动端抽屉**：提供独立按钮开关，阅读时不挡内容。

### 2. 文章目录（TOC）增强

- 自动根据文章内的 `## / ###` 标题生成目录。
- 点击目录项平滑滚动定位，并在移动端自动收起抽屉。
- 滚动阅读时高亮当前章节。

### 3. 主题与背景系统完善

- 主题色与背景图状态可持久化保存。
- 支持 **"仅背景图"模式**（减少主题色遮罩），提升图片背景的观感。

### 4. 返回顶部按钮

当你向下滚动到一定距离，会出现 **返回顶部** 按钮：

- 桌面端支持 hover 动画。
- 点击后平滑滚动回到顶部。

## 接下来会做什么？

v1.0 完成后，我会继续在这个站点上做两件事：

1. **持续写作**：把学习和项目沉淀成可复用的内容。
2. **持续迭代体验**：比如更丰富的文章组件、更完善的搜索、以及更细腻的动效。

如果你偶然来到这里，希望这些内容能对你有所帮助。

最后，欢迎通过侧边栏的 GitHub / 邮箱联系我交流。
